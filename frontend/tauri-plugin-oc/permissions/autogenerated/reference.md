## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-open-url`
- `allow-sign-in`
- `allow-sign-up`
- `allow-registerListener`
- `allow-removeListener`
- `allow-get-fcm-token`
- `allow-show-notification`
- `allow-svelte-ready`
